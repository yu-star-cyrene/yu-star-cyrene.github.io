#命令小指南

www-data 低权限用户(默认用户)

##Docker：

docker ps 确认容器名字

docker exec -it "容器名字" /bin/bash 进入容器

cd /var/www/html 定位源码 (url)

##mysql：

show tables from 数据库 查查数据库内的表

select * from 数据库.表; 查数据库中的某一个表
